# ❤️ Heart Disease Diagnostic Dashboard using MySQL, Tableau & Flask

## 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early diagnosis and proper analysis of patient health data can help healthcare professionals identify high-risk patients and make better decisions.

This project analyzes heart disease patient data using **MySQL**, **Tableau**, and **Flask**. The data is stored in a MySQL database, connected to Tableau for visualization, and finally embedded into a Flask web application to provide an interactive dashboard for users.

---

# 🎯 Problem Statement

Healthcare organizations generate a large amount of patient data every day. Analyzing this data manually is time-consuming and inefficient.

The objective of this project is to transform raw heart disease data into meaningful visual insights that help:

- Identify high-risk patients
- Analyze important health indicators
- Support healthcare decision-making
- Increase awareness about heart disease risk factors

---

# 🎯 Objectives

- Store heart disease data in MySQL.
- Perform SQL operations for data management.
- Connect MySQL with Tableau.
- Create interactive visualizations.
- Build a dashboard and story in Tableau.
- Integrate the Tableau dashboard into a Flask web application.
- Present healthcare insights in an interactive way.

# 📊 Key Insights

From the analysis, the following observations were identified:

- Middle-aged and older adults show a higher occurrence of heart disease.
- Certain chest pain types are strongly associated with heart disease.
- Patients with exercise-induced angina are more likely to have heart disease.
- Cholesterol and resting blood pressure vary significantly among patients.
- Interactive dashboards make it easier to explore healthcare data and identify trends.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/likithasatti/heart-disease-analysis.git
```

Go to the project folder:

```bash
cd heart-disease-analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000/
```
